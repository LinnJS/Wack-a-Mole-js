# Getting Started
- npm or yarn install
- npm or yarn start
- open `localhost:8000`

## Additional Scripts
- ESLint: `npm run lint`
- ESLint with formatting: `npm run lint:fix`
- Development mode with HotReloading: `npm run dev`
